# ProD FrontEnd Conventions

ProD 웹프론트엔드의 개발 업무 규칙를 모아 놓았습니다. **규칙을 지키는 것은 의무가 아니나 가급적 서로간 원활한 커뮤니케이션을 위해 지키도록 합시다.**

원활한 커뮤니케이션을 위함은 물론, 이른바 **클린 코드**를 위하여서도 업무 규칙은 필요한 사항이기 때문에 모두 자유롭게 의견을 제시하면 좋을 거 같습니다.
